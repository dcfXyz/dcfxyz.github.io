memset函数

By dcf_xyz

发表于 2015-05-20

文章目录

memset是库函数，被用来格式化数组。
memset的函数原型：
void* memset(void * buf, int ch, size_t count);

  • 第一个参数：需要格式化的区间的首地址
  • 第二个参数：用来填充的元素
  • 第三个参数：需要填充的字节数

memeset是按照字节来进行填充的，也就是将所在区间的每一个字节用第二个参数对应的ascii码值进行替换。一个字节一个字节地格式化，所以能方便地将数组格式化为0，-1，true， false。

刚开始不了解的话，很容易理解为填充为固定的值，比如

这样的话其实不会的到想要的结果，由于他是将其格式化为第二个参数所对应的ascii码值，所以对应的每一个数字就变成了0X01010101也就是16843009。

它能被用来将数组格式化为true和false，是由于true和false是本身就是一个字节；
用0来填充是因为0就是00000000，所以你怎么读都是0；而能够用来填充为-1，则是由于-1在计算机内的存储是按照补码来存储的-1，也就是11111111所以也是没有问题的。

在c++中还有了一个用来格式化一个区间的函数，即algorithm中的fill_n
fill_n的函数原型：
iterator fill_n(iterator start, size_t n, const TYPE& val);

  • 第一个参数：待格式化区间首元素迭代器
  • 第二个参数：需要填充的元素个数
  • 第三个参数：用来填充的元素

在用来格式化容器的同时，也可以用来格式化数组。
