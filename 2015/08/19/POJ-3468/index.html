POJ-3468 （线段树 区间更新）

  • 解题报告
ACM
文章目录

    1. 1.1. 面对的问题
    2. 1.2. 解决的方法
    3. 1.3. 具体的操作
  2. 2. 代码

题目链接：poj3468 - A Simple Problem with Integers

思路

线段树的应用，区间更新 和 区间查询，比较困难的就是区间更新了。

面对的问题

线段树之所以能够快速进行区间操作，是因为它的每一个区间分解得到的终止区间都是 $O(\log n )$ 级别的，但是如果每一次区间更新都将操作进行到叶子节点，那么操作就会达到 $O(n\log n)$级别。

解决的方法

因为很多底层的信息是不会用到的，所以我们在更新时，也像区间查询那样更新到 “终止区间”，将更新信息记录下来即可，这样操作又回到了 $O(\log n)$ 级别，只有当我们进行查询是需要用到这个区间的信息的时候，再利用 push_down 操作将当前信息更新到下一层。

具体的操作

对这道题来说每个节点需要维护 和值 和 增量 两个变量。

  • 建树，没有什么差别
  • 更新，未到达 “终止节点” 时，更新 和值； 到达后，更新 增量
  • 查询，未到达 “终止节点” 时，增量 不为 0 进行 push_down 操作，到达后，返回当前 和值。
